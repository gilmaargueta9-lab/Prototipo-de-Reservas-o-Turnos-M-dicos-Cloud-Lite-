# Prototipo de Reservas o Turnos Médicos en la Nube (Lite)

Proyecto desarrollado como parte de la Estancia Profesional ESIT.

## Cliente
Clínica Integral Santa Salud (CISS)

## Objetivo
Desarrollar un prototipo cloud lite para la gestión de citas médicas.

## Estructura del repositorio
- /frontend: interfaz del prototipo
- /backend: lógica de negocio y validaciones
- /docs: documentación del proyecto
- /diagrams: diagramas de análisis y diseño
- /bitacora: registro semanal del proyecto
